# Shortest Path Finder
Pathfinder is a navigation tool designed to determine the most efficient route between any two selected points on Quang Trung Ward in Dong Da District, Hanoi, Vietnam.
With just a few clicks, users can select two locations within the designated boundary area. Shortest Path Finder will then quickly compute and display the optimal route, ensuring smooth and effective navigation across the mapped zone.

# How to Use Pathfinder
Start by running the backend.py script located in the src directory.
Next, open the map2.html file found in the data folder using your web browser to find the shortest path between point A and point B.
For the Admin, open map2_flood.html and map2_traffic.html to simulate real life scenarios on the map. 
Enjoy seamless navigation using Shortest Path Finder’s interactive mapping interface.
